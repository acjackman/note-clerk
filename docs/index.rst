Command-line application for mantaining plaintext notes.

Installation
------------

To install Note Clerk, run this command in your terminal:

.. code-block:: console

   $ pip install note-clerk


Usage
-----

Hypermodern Python's usage looks like:

.. code-block:: console

   $ note-clerk [OPTIONS]

.. option:: --version

   Display the version and exit.

.. option:: --help

   Display a short usage message and exit.
